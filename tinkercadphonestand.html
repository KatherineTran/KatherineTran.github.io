Tinkercad Project #1: Phone Stand

In trimester 3, our computer science class worked with 3D printing using the design website, Tinkercad. This is a user friendly site with lessons about designing, making holes, and other need-to-knows about the designing a 3D object. There was also a public database of other users' shapes and databases that can be copied and tinkered as your own project. To familiarize myself with the controls, one of the lessons was to create a phone stand based on the given instructions

This project killed two birds with one stone to introduce students to tinkercad while creating a practical object that can be printed and used in real life.

Using inspiration from the internet, I deviated from the original design to create an elephant phone stand with my initials on the ears. While this was a useful phonestand, it also included a small hole inside the elephant body that could hold small objects and pencils.
