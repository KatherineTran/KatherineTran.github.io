Tinkercad Project

In trimester 3, our computer science class worked with 3D printing using the design website, Tinkercad. This is a user friendly site with lessons about designing, making holes, and other need-to-knows about the designing a 3D object. There was also a public database of other users' shapes and databases that can be copied and tinkered as your own project.

One of our first projects using tinkercad was to use basic shapes such as cubes, roof, semi-circle, circle, etc. to create a house. During the project, I practiced the skills of making holes inside the house for the rooms, changing the colors of the windows and shapes, and making a floor plan. I also practiced copying and uploading other user's shapes. I utilized the skills of using the grid scale button and changing the dimensions in order to fit into each other to make it water tight with no gaps.
